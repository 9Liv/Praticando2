Questionário


Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.
R.: Entregar arquivos estáticos como HTML, CSS e imagens. Exemplo: Apache.


Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?
R.: Sua principal característica é ser leve e eficiente, por isso, é utilizado em projetos de alto desempenho e proxy reverso.


Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?
R.: HTTP: Protocolo para transferência de páginas web (sem criptografia).
HTTPS: Versão segura com criptografia SSL/TLS. É recomendado porque protege os usuários, e a integridade de seus dados, além de garantir a autenticidade da comunicação dos servidores.


Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?
R.: As portas são canais de comunicação para serviços e aplicações específicas de um dispositivo. A porta 443 é a padrão para HTTPS, com isso, garante privacidade, segurança e autenticidade.


Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.
R.: O endereço IP é o número identificador único de um dispositivo conectado na rede, é essencial para identificação de cada aparelho e assim, permitir o transporte de informações correto e garantir ampla comunicação em escala global.


Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.
R.: Hospedagem VPS funciona como um servidor virtual com mais controle e desempenho, cada máquina virtual isolada possui suas próprias designações. É indicada quando a hospedagem compartilhada não consegue dar conta do crescimento do site, quando existe a necessidade de hospedar muitos sistemas em um mesmo servidor, há urgência de estabilidade e segurança na aplicação.


Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.
R.: Possui configuração simples, https automático, suporta proxy, http/2 e http/3, é moderno e seguro.


Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.
R.: Microsoft IIS, ele é otimizado para a plataforma NET(C#), facilita o desempenho pois apresenta integração nativa.


Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.
R.: Modelo onde o cliente (navegador ou app) envia requisições a um servidor, que responde com os dados/processamento solicitado.


Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.
R.: É um servidor de aplicação Java, indicado para aplicações web baseadas em servlets e JSP.


Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.
R.: O servidor web, entrega algo estático, que não sofre mudanças recorrentes, e apresenta a mesma estrutura para todos os usuários, como uma página da web, arquivos e imagens, já o servidor de aplicação lida com o dinamismo, e lógica, fornece atualizações e executa códigos para isso.


Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.
R.: Domínio é o nome que representa o IP, que identifica o servidor.


Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?
R.: A hospedagem em nuvem garante escalabilidade alta, escalonamento e disponibilidade fácil.


Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.
R.: Apache Tomcat (Java) e Microsoft IIS (.NET).


Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?
R.: HTTPS protege dados com criptografia enquanto HTTP é vulnerável a interceptação e ataques.


Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.
R.: Quando o site cresce e precisa de mais recursos e controle do servidor.


Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.
R.: IP identifica o dispositivo na rede já a porta identifica o serviço ou aplicação dentro do dispositivo.


Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?
R.: Garantir segurança e privacidade dos dados trocados com os usuários.


Questão 19: Cite dois exemplos de servidores web e explique suas principais características.
R.: Apache: robusto e configurável. Nginx: leve, rápido e ótimo para alta concorrência.


Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.
R.: Vários sites dividem os mesmos recursos de um servidor é utilizada para sites pequenos ou iniciantes.


Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.
R.: Executa lógica de negócio e gera conteúdo dinâmico; servidores estáticos só entregam arquivos pronto.


Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.
R.: Gunicorn: servidor WSGI que executa aplicações Python e gerencia múltiplos processos.


Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?
R.: Servidor de aplicação Java EE para executar aplicações empresariais Java.


Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?
R.: Usada para controle do protocolo FTP, que serve para transferência de arquivos.


Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.
R.: Ele suporta arquivos de configuração Apache (.htaccess), facilitando migrações e mantendo compatibilidade.


Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?
R.: Servidor web da Microsoft, usado principalmente para aplicações .NET em ambientes Windows.


Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.
R.: Ambiente para executar JavaScript no servidor, usado para aplicações em tempo real e APIs rápidas.


Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.
R.: Servidor de aplicação Java EE para executar aplicações Java corporativas.


Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?
R.: Servidores web para aplicações Ruby on Rails, gerenciam processos e conexões.


Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?
R.: Impacta custo, desempenho, segurança e escalabilidade, garantindo que o projeto funcione bem e cresça conforme necessário.
